Welcome to Ross' FCC Timestamp Service!

The purpose of this service is to provide both a unix and natural date to a user see the instructions below for how to use it

Usage:

Add "/(Your Unix or Natural Date)" to the end of the url for this page to receive a json object containing the unix and natural form of your input date. (See sample object below)
{ unix: (Unix Date of Input), natural: (Natural Date of Input) }

This endpoint accepts both unix times and a time in natural form. If an invalid time is entered the endpoint will return the json object with both parameters as null. Note that for inputting a time in natural form it is preferred that if a time is input in natural form that the day, then the month then the year is provided (in that order). It is also necessary to have these numbers/strings space or comma separated.
